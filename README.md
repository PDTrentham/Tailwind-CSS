This project was for learning how to install and use TailwindCSS.

During this project I have used Tailwind's classes to customise the layout and colours the website use to make a home page such as arranging the top into a bold title, creating a navigation to the side of the top and then colouring text with features such as setting text-emerald-100. This sets the text to an emerald colour.
Text size can also be changed by setting text-4xl which sets the text to use Tailwind's extra large size and then makes that four times larger.

Then I created a gallery page with a background gradient going from a dark emerald to a lighter emerald with bordered images in a row along the page.


This can be viewed in browser by opening "https://pdtrentham.github.io/Tailwind-CSS/" and then clicking "home".
